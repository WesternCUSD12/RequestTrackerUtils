üéØ Project Overview

Request Tracker Utils is a Django-based web application designed to simplify asset lifecycle management and student device tracking. This system integrates with Request Tracker (RT) to provide enhanced workflows for creating assets, generating labels, managing device check-ins, and conducting device audits.

  • Asset Creation: Streamlined workflow for creating RT assets with automatic tag sequencing
  • Label Generation: Print QR code and barcode labels for assets (publicly accessible)
  • Device Tracking: Check-in/check-out system for student device management
  • Audit Workflows: CSV-based device verification and audit sessions
  • Student Management: Track student device assignments and RT user mappings
